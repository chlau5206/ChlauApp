Highlighted Projects

Exchange Rate Viewer — API Constraints & Data Transformation

A lightweight web application designed to demonstrate safe API integration and controlled data handling. The free exchange‑rate API used in this project has strict limitations: it allows only a few requests per day and provides rates exclusively in EUR as the base currency. To work within these constraints, I built server‑side logic that limits API calls, caches the results, and converts the base currency from EUR to USD. The current public demo displays transformed data only, without live API calls, to avoid exposing the API key and to stay within the API’s request limits.

CHS → CHT ePub Conversion Utility (C# Globalization Project)

A custom Windows desktop application built in C# to convert Simplified Chinese (CHS) ePub books into Traditional Chinese (CHT). Unlike simple code‑page converters, CHS→CHT requires full character mapping, contextual replacements, and transformation of the internal XHTML files inside the ePub archive. Existing online converters handle plain text but do not support ePub formats, so I built this tool to solve a real personal need: converting my CHS ePub books into CHT versions when no suitable software existed. The utility extracts the ePub structure, applies character‑mapping logic to all relevant content files, and rebuilds the ePub package. This project demonstrates practical C# skills, understanding of Unicode and encoding systems, and the ability to design tools that solve real-world globalization and text‑processing challenges.

Message Board / BoardDemo — Web Development & Dual‑Database Architecture

A Flask‑based message board demonstrating clean web development practices, form handling, SQLAlchemy models, and a secure dual‑database architecture. The demo uses an in‑memory database for safe public interaction while keeping production data isolated. This project highlights blueprint organization, backend logic, data validation, and the ability to design systems that are modular, maintainable, and safe to expose publicly.
