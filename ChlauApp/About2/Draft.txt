Highlighted Projects
Exchange Rate Viewer — API Constraints, Data Transformation, and Safe Public Demo
A lightweight web application designed to demonstrate safe API integration and controlled data handling. The free exchange‑rate API used in this project has strict limitations: it allows only a few requests per day and provides rates exclusively in EUR as the base currency. To work within these constraints, I built server‑side logic that limits API calls, caches the results, and converts the base currency from EUR to USD.
The current public demo displays transformed data only, without live API calls, to avoid exposing the API key and to stay within the API’s request limits. While not a full ETL pipeline, the project includes meaningful data transformation and controlled ingestion steps that reflect real‑world API handling patterns.

Encoding & Code Page Conversion Tool — C# Globalization Utility
A Windows desktop application built in C# to convert text between different code pages and Unicode formats. Designed to demonstrate understanding of character encoding, globalization, and Windows desktop development. The tool showcases practical C# skills, UI design, and the ability to build utilities that solve real‑world data handling problems.

Message Board / BoardDemo — Web Development & Dual‑Database Architecture
A Flask‑based message board demonstrating clean web development practices, form handling, SQLAlchemy models, and a secure dual‑database architecture. The demo uses an in‑memory database for safe public interaction while keeping production data isolated. This project highlights blueprint organization, backend logic, data validation, and the ability to design systems that are modular, maintainable, and safe to expose publicly.
